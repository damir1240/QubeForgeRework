# QubeForge Rework

**QubeForge Rework** — это обновленная и оптимизированная версия оригинального проекта [QubeForge](https://github.com/neurochelplus/QubeForge) от команды **Нейрочел+**.

## О проекте
Это воксельная песочница, вдохновленная классическими играми жанра. В рамках реворка была проведена значительная работа по улучшению архитектуры, оптимизации производительности и очистке кодовой базы.

### Основные изменения:
- **Архитектура**: Переход на событийно-ориентированную модель (Event Bus) для снижения связанности компонентов.
- **Система ввода**: Централизованная `InputSystem`, поддерживающая как десктопное управление, так и мобильные устройства.
- **Производительность**: Оптимизация рендеринга и управления чанками.
- **Чистота кода**: Удаление устаревших зависимостей и легаси-кода.

## Технологии
- **Vite** — быстрая сборка и разработка.
- **Three.js** — 3D рендеринг.
- **TypeScript** — типизация и безопасность кода.

## Разработка
Для запуска проекта локально:

```bash
npm install
npm run dev
```
