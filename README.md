# QubeForge Rework

**QubeForge Rework** — это независимый реворк оригинального проекта [QubeForge](https://github.com/neurochelplus/QubeForge) от **Нейрочел+**.

## О проекте
Данный проект представляет собой независимое видение игры от пользователя **damir1240**. Основными целями реворка являются значительное улучшение производительности и выпуск регулярных контентных обновлений.

**Важно:** Этот проект развивается отдельно от оригинала. Обратная совместимость (сохранения, моды и т.д.) между оригинальным QubeForge и данным реворком не гарантируется.

## Технологии
- **Vite** — быстрая сборка и разработка.
- **Three.js** — 3D рендеринг.
- **TypeScript** — типизация и безопасность кода.

## Разработка
Для запуска проекта локально:

```bash
npm install
npm run dev
```
